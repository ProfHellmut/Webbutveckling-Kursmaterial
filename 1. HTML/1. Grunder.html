Kurs: Webbutveckling 2 Lärare: Markus Nordin Syfte: Kortfattat täcka grunderna för HTML HTML: HTML (Hypertext Markup Language) används för att skapa strukturen för webbsidor på internet och är en av tre teknikerna vi använder för att skapa webbsidor(HTML, CSS och JavaScript). För att enkelt få en bild av hur dessa tre tekniker fungerar med varandra så kan man se på det som att HTML är skelettet, CSS är huden, ögonfärgen, hårfärgen (det estetiska) och JavaScript är musklerna och nerverna(det som gör kroppen dynamisk och funktionell). Med HTML skapar vi då som sagt strukturen för en webbsida, och det vi menar med det är att vi placerar ut element för exempelvis paragrafer, länkar, bilder, headers och annat vi kan tänkas behöva i den ordning den ska vara. Ordningen man placerar sina element i är viktig och man bör tänka på var de hör hemma. Ta exempelvis att vi skapar en sida med information om skolan. Här börjar vi med att placera ut en header med skolans namn samt en bild. Därefter skapar vi ett navigationsfält för länkar till andra sidor, sedan har vi huvudinnehåll och avslutningsvis så har vi en footer med telefonnummer och address till skolan. Det vill säga, vi skapar strukturen från toppen till botten. Taggar och element: Taggar och element är två ord ni kommer höra ofta, och ibland kan det bli svårt att hålla koll på vad som är vad. En tagg omges av vinkelparenteser ("<" och ">") med en eller flera bokstäver i mitten som beskriver dess funktionalitet. Exempelvis så är "

Det finns både lokala och globala attribut. Med globala attributer menar man attributer som kan tillämpas på alla element som exempelvis attributen "id" och "class", exempel:

Denna paragraf har en class

. Detta element har nu klassen "myClass" som vi kan använda oss av när vi exempelvis vill skriva specificerad CSS kod för elementet (Detta går vi igenom mer när vi börjar med CSS). Med lokala attribut menar vi element-specifika attribut och som exempel tar vi ""-elementet som exempel. Med "" skapar vi länkar, antingen externa länkar eller lokala (mellan olika dokument) länkar. För att göra detta måste vi specificera en address länken ska leda till och detta gör vi med attributet "href". Vill vi skapa en länk till en extern webbsida behöver vi inkludera URL:en för denna och vill vi skapa en länk till ett lokalt dokument så skriver vi sökvägen till det dokumentet. Exempel för extern webbaddress: Google. Exempel för lokal: Till andra sidan!. Vid lokal länkning bör man vara noga med att inkludera filtypen i namnet(".html").
